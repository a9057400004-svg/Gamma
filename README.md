# МАГМА — сайт для Vercel

Премиальный сайт студии покраски и реставрации.

## Что внутри

- Next.js 14 + TypeScript
- адаптивный премиальный дизайн
- галерея работ
- форма заявки
- отправка заявки в Telegram
- контакты мастерской
- готовность к деплою на Vercel

## Переменные окружения для Vercel

В настройках проекта Vercel нужно добавить:

```env
TELEGRAM_BOT_TOKEN=токен_бота_от_BotFather
TELEGRAM_CHAT_ID=-1004447953221
```

После добавления переменных нужно сделать Redeploy.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
